# Setup
- `git clone https://github.com/OpenKinect/libfreenect2.git'
- `cd libfreenect2`
- Max and linux follow guide on github
- `cmake .. -DCMAKE_INSTALL_PREFIX=/usr/local/`
- `make && sudo make install`

- `git clone https://github.com/opencv/opencv.git`
- `mkdir build`
- `cd build`
- `cmake ..`
- `make && sudo make install`
